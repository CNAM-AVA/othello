# othello
Jeu de l'othello en C via les sockets

## Compilation

### Linux

Compiler le programme avec make:

`>make`

Lancer le programme:

`>./othello <port>`

### Mac

1. Installer XQuartz
2. Installer socat
3. Compiler le programme avec `./docker/build.sh`
4. Lancer le programme avec `./docker/run.sh <port>`
